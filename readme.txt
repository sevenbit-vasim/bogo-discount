Automatically applies a BOGO discount for every second item in the cart.

== Description ==
This plugin automatically applies a Buy One Get One Free (BOGO) discount for every second item in the cart. Simply activate the plugin, and the BOGO discount will be applied during the checkout process.

== Installation ==
A. Upload via WordPress Dashboard

Log in to your WordPress admin dashboard.
Navigate to Plugins > Add New.
Click on the Upload Plugin button.
Choose the .zip file of your custom plugin or directly upload the .php file if you didn't compress it.
Click Install Now.
Once the plugin is installed, click on Activate Plugin.

B. Upload via FTP
Use an FTP client (like FileZilla) to connect to your web server.
